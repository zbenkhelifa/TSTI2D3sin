Créer un compte
GitHub

Guide interactif étape par étape — même sans expérience

Gratuit
5 minutes
8 étapes
Étape 1 / 8

Accéder au site GitHub

Ouvre ton navigateur (Chrome, Firefox, Edge…) et tape l'adresse suivante dans la barre d'URL :

github.com
🔒 github.com
Tu arrives sur la page d'accueil de GitHub

Une fois sur la page, cherche le bouton Sign up en haut à droite et clique dessus.

Étape 2 / 8

Entrer ton adresse e-mail

La première chose demandée est ton adresse e-mail. Saisis-la dans le champ, puis clique sur Continue.

✅ Utilise une adresse à laquelle tu as accès — tu devras confirmer ton inscription par e-mail. Une adresse Gmail ou scolaire convient parfaitement.

GitHub vérifie que l'adresse n'est pas déjà utilisée. Si c'est le cas, utilises-en une autre ou connecte-toi à ton compte existant.

Étape 3 / 8

Choisir un mot de passe

Crée un mot de passe sécurisé. GitHub recommande au moins 15 caractères, ou 8 caractères minimum avec des chiffres et symboles.

⚠️ Note ton mot de passe dans un endroit sûr (gestionnaire de mots de passe, carnet…). Il ne peut pas être récupéré si tu l'oublies.

GitHub t'indique en temps réel si le mot de passe est assez solide — attends le feu vert avant de cliquer Continue.

Étape 4 / 8

Choisir un nom d'utilisateur

Ton username est ton identité publique sur GitHub. Il apparaîtra dans toutes tes URLs :

github.com/ton-username
💡 Choisis quelque chose de professionnel : prenom-nom ou prenomnom. Pas d'espaces ni de caractères spéciaux (seul le tiret - est autorisé).

Si le nom est déjà pris, GitHub te propose des alternatives. Une fois satisfait, clique sur Continue.

Étape 5 / 8

Préférences e-mail

GitHub demande si tu veux recevoir des e-mails de communication sur les nouveautés du produit.

Tape y pour oui ou n pour non, puis clique sur Continue.

ℹ️ Ce choix n'a aucun impact sur les e-mails de sécurité ou de vérification — ceux-ci arrivent toujours quoi qu'il arrive.
Étape 6 / 8

Vérification humaine

GitHub affiche un CAPTCHA — un petit test visuel pour s'assurer que tu n'es pas un robot.

Suis les instructions à l'écran (cliquer sur des images, résoudre un puzzle…). Une fois le test réussi, le bouton Create account devient cliquable.

💡 Si le puzzle ne se charge pas, recharge la page ou essaie dans un autre navigateur.
Étape 7 / 8

Vérifier ton e-mail

GitHub envoie un code de vérification à 6 chiffres à l'adresse que tu as indiquée. Ouvre ta boîte mail et cherche un message de noreply@github.com.

Copie le code et colle-le dans les cases affichées sur le site. Ton compte est instantanément activé.

📬 Tu ne vois pas l'e-mail ? Vérifie le dossier Spam ou Indésirables. Le code expire au bout de 15 minutes.
Étape 8 / 8

Personnaliser ton profil

GitHub propose quelques questions sur ton expérience et tes objectifs. C'est facultatif : tu peux tout remplir ou cliquer sur Skip personalization.

Tu arrives ensuite sur ton dashboard — le tableau de bord de ton compte. C'est tout !

🎉 Compte créé avec succès. Tu peux maintenant créer ton premier dépôt, explorer des projets open source et collaborer avec d'autres développeurs.
